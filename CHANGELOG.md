# 更新日志

### 🎉 首次发布

#### 新增功能
- ✅ **文件管理**
  - 文件上传和下载
  - 文件列表展示
  - 文件删除功能
  - 下载统计

- ✅ **分享功能**
  - 分享链接生成

  - PC 端优化布局
  - 手机端适配

  - 流畅的用户交互

- ✅ **技术特性**
  - Hash 路由分享页面
  - 时间戳防缓存机制
  - 跨浏览器兼容性
  - 智能复制方案（Clipboard API + 备用方案）

- Spring Boot 2.7.x 框架

### 🚀 快速开始

**启动后端：**
```bash
cd backend
mvn spring-boot:run
```

**启动前端：**
```bash
npm install
npm run dev
```
